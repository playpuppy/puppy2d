Puppy

Puppy is a programming language designed for first programming courses.

At a glance, Puppy is:

  • Simple: Puppy is a restricted subset of Python 3, which can easily be run on a Web browser.
  • Familiar: Puppy programs can be executed directly in a Python (3.6+). It can also be edited using standard Python editors.
  • Fun: Puppy program runs with Physics simulation (using puppy2d). Even small code enables interesting behaviors.
